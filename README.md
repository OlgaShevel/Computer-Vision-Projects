### Annotation_Cuboids_without_CVAT
Разметка кубоидов (псевдо-3D) без использования фреймворка CVAT
<br><br>(Cuboid Annotation (pseudo-3D) without CVAT)

### !Adversarial_attacks!.ipynb
Состязательные атаки метода FGSM на признаковую составляющую неразмеченного датасета изображений. Достаточно возмущения градиента, равного ε = 0,01, чтобы точность классификации упала почти вдвое против незараженного датасета.
<br><br>(Adversarial attacks using the FGSM method on the feature representation of an unlabeled image dataset. A gradient perturbation with ε = 0.01 is sufficient to reduce classification accuracy by nearly half compared to the clean (unperturbed) dataset).</br>

### !Classification_clustering_pre_trained.ipynb
Способ выбрать оптимальные предобученные модели для классификации неразмеченных изображений
<br><br>(A method for selecting optimal pre-trained models for classification of unlabeled images).</br>

### !Comparison_of_pre_trained_models.ipynb
Сравнение предобученных моделей из фреймворков Keras и Pytorch на задаче классификации изображений
<br><br>(Comparison of pre-trained models from the Keras and PyTorch frameworks on the image classification task).</br>

### !DICOM_reconstr_3D.ipynb
Мультипланарная реконструкция (MPR) анатомических проекцией черепа человека и реконструкция 2D -> псевдо-3D из файлов DICOM с использованием единиц радиоплотности Хаунсфилда (HU).
<br><br>(Multiplanar reconstruction (MPR) of anatomical projections of the human skull, and 2D → pseudo-3D reconstruction from DICOM files using Hounsfield Units (HU) for radiodensity quantification).</br>

### !GradCAM_Archiv_Nerworks.ipynb
Применение алгоритмов GradCam/GradCam++ при обучнии архивных сверточных нейросетей AlexNet (2012), NiN и ZFNet (2013).
<br><br>(Application of Grad-CAM/Grad-CAM++ algorithms in the training of legacy convolutional neural networks: AlexNet (2012), Network-in-Network (NiN), and ZFNet (2013)).</br>

### !semseg_DeepLab2+ResNet101.ipynb
Эксперимент: семантическая сегментация с целью получения 100%-ной воспроизводимости пикселей при помощи предобученных DeepLabv2 + ResNet101
<br><br>(Experiment: Semantic segmentation aimed at achieving 100% pixel-level reproducibility using pre-trained DeepLabv2 + ResNet101).</br>

### !Сontours_in_OpenCV.ipynb
Эксперимент: Многоклассовая классификация изображений по контурному краю Canny Edge Detection на примере InceptionV3
<br><br>(Experiment: Multi-class image classification based on contour features extracted via Canny Edge Detection, using InceptionV3 as a case study).
